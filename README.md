# markov_chains
Небольшой модуль, позволяющий характеризовать дискретный Марковский процесс по матрице переходов.



# пример использования
В файле markov.py.

$$
\begin{equation}
\left\{
\begin{aligned}
x + 2y - z &= 5 \\
x - 3y + z &= 0 \\
-3y + z &= 0 \\
-3x + 2y &= 7 \\
\end{aligned}
\right.
\end{equation}
$$
